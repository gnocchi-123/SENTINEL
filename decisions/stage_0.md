# Stage 0 — 사양 검토 및 충돌 확인

**날짜:** 2026-05-30  
**목적:** `sentinel_knowledge_v0.3.md`와 `SENTINEL_instructions_v0.3.md` 숙지 및 사용자 지시와의 충돌 여부 확인. 코드 미작성.

---

## 확인된 주요 가정 (확정 사양)

### 리밸런싱 방식
- 매월 첫 번째 거래일에 **점검**
- 신호가 전월 대비 바뀐 경우에만 **매매 실행**
- **무신호 = 무행동**: 국면 필터 + 섹터 순위가 동일하면 거래 없음

### 체결 타이밍 (룩어헤드 금지 핵심 규칙)
- **신호 산출:** 점검일(M월 첫 거래일) 직전 월말(M-1월 마지막 거래일) 종가까지의 데이터만 사용
- **체결:** 당월(M월) 첫 거래일 종가
- 예: 12월 말 데이터로 신호 → **1월** 첫 거래일 체결 ("당월" = 점검월)
- assert + 단위 테스트로 룩어헤드 금지 보장 필요

### 모듈 구조 (4개 + 워치독 오버레이)
1. **시장 국면 필터:** SPY ≥ 200일 MA AND SPY 12개월 수익률 > BIL → 둘 중 하나 미충족 시 전량 방어자산
2. **섹터 상대모멘텀:** 3·6·12개월 총수익률 단순평균 순위 → 상위 N개 보유 (기본 N=3)
3. **포지션 크기:** 균등배분(기본), 변동성 역가중(토글)
4. **청산 조건:** 섹터 순위 탈락 또는 국면 필터 OFF 전환 시에만. 고정 손절 없음.
5. **워치독(오버레이, 기본 OFF):** SPY 252일 고점 대비 −10% AND VIX > 28 → 주식분 50% 부분 청산, 재진입은 다음 정규 월초 점검으로만

### 유니버스
- 섹터 ETF: XLK, XLV, XLF, XLE, XLI, XLY, XLP, XLU, XLB, XLRE, XLC (11개)
- 반도체 후보: SOXX (기본), SMH (토글) — 11개 섹터와 동일 모멘텀 순위 풀에서 경쟁
- 방어자산: AGG (기본), BIL·IEF (토글)
- 기준선: BIL (무위험 수익률 비교용)
- 국면 필터: SPY 일봉 (200일 MA용)
- 워치독 보조: ^VIX 일봉

### 데이터 처리 규칙
- 늦게 상장된 ETF(XLRE 2015, XLC 2018 등): 상장 전 기간 NaN 처리 → 해당 시점 순위 풀에서 자동 제외
- 지수 스플라이싱 미적용 (v1 제외, 향후 과제)
- 재실행 시 캐시로 결정론적 재현 보장

### 파라미터 관리
- 모든 파라미터는 `params.yaml`에서 관리. 코드 내 매직넘버 하드코딩 금지.
- 주요 파라미터: `top_n=3`, `ma_window=200`, `momentum_windows=[3,6,12]`, `cost_bps=5`, `defense_asset=AGG`, `semiconductor=SOXX`
- 워치독: `watchdog_enabled=false`, `drawdown_threshold=-0.10`, `high_lookback_days=252`, `vix_threshold=28`, `derisk_fraction=0.50`, `refire=false`, `reentry=next_regular`

### 거래비용
- 편도 5bp (0.05%). 0으로 토글 가능.

## 주요 결정 사항

| 결정 | 근거 |
|---|---|
| 체결 = 당월 첫 거래일 (익월 아님) | 한 달 추가 지연 시 하락 방어 반응 속도 저하 (우선순위 1번 위반) |
| 워치독 기본 OFF | 휩쏘 위험(V자 급락 시 바닥 매도) 존재. 백테스트로 느린 하락/V자 유형별 검증 후 채택 결정 |
| SOXX 특별 로직 없음 | 순위 풀 내 동일 규칙 적용. SOXX/XLK 중복 허용, 별도 처리 금지 |
| 개별 종목 고정 손절 없음 | 섹터 순위 탈락·국면 필터 OFF만을 청산 트리거로 사용. 단기 노이즈 허용 |
| 무신호 = 무행동 (비중 재조정도 없음) | 회전율 최소화, 감정 개입 차단 |
